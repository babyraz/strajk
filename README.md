som del av kursen Fullstack webbapplikation med TypeScript.

I appen kan man kan välja datum, tid, banor, spelare och skostorlekar, och sen få en bekräftelse med pris och bokningsnummer. Jag försökte följa Figma-skissen så gott det gick och få allt att kännas smidigt på mobil.

Jag valde att använda localStorage i stället för Zustand eftersom appen är ganska liten och jag egentligen bara behövde spara senaste bokningen. 

Under arbetet använde jag AI (ChatGPT) som hjälp när jag fastnade, bland annat för att lösa ett knepigt problem där iOS inte lät mig radera siffror i nummerfält. Jag fick tipset att hantera värdena som strängar i stället för nummer, och sedan konvertera dom senare, vilket löste problemet.
